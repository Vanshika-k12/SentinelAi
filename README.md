# 🛡️ SentinelAI — Digital Public Safety Intelligence Platform

AI-powered platform to detect, disrupt, and respond to digital fraud, scam operations, and cybercrime across India.

## Modules

- **Citizen Fraud Shield** — AI fraud detection for suspicious messages and screenshots
- **Fraud Network Graph** — Visualise coordinated fraud rings and money mule networks
- **Geospatial Crime Map** — Interactive India cybercrime heatmap
- **Law Enforcement Copilot** — AI assistant for investigators and command centres

## Tech Stack

- Frontend: Streamlit
- AI: Google Gemini 2.0 Flash
- Graph: NetworkX + Matplotlib
- Maps: Folium
- OCR: EasyOCR
